หลักสูตร - คำสั่ง while loop ในภาษา C
หน้าแรก เกี่ยวกับ หลักสูตร ตัวอย่างโปรเจค ติดต่อ

1. แนะนำคำสั่ง while loop

คำสั่ง while เป็นคำสั่งควบคุมโครงสร้างแบบวนซ้ำ ที่จะทำซ้ำคำสั่งภายในบล็อกจนกว่าเงื่อนไขจะเป็นเท็จ

2. โครงสร้างของ while loop

while (เงื่อนไข) { // คำสั่งที่ต้องการให้ทำซ้ำ }

3.ตัวอย่างการใช้ while loop

1. ตัวอย่างพื้นฐาน (การพิมพ์ค่าออกทางหน้าจอ)

โค้ดนี้จะแสดงค่าตัวเลขจาก 0 ถึง 4:


            int i = 0;
            while (i < 5) {
                printf("%d\n", i);
                i++;
            }

2. การใช้ while loop ในการรับค่าจากผู้ใช้

โค้ดนี้รับค่าจากผู้ใช้จนกว่าจะป้อนเลข 0:


            int num;
            printf("ป้อนตัวเลข (0 เพื่อออก): ");
            scanf("%d", &num);
            
            while (num != 0) {
                printf("คุณป้อน: %d\n", num);
                printf("ป้อนตัวเลขอีกครั้ง: ");
                scanf("%d", &num);
            }

3. การประยุกต์ใช้ while loop ในการประมวลผลข้อมูล

โค้ดนี้คำนวณผลรวมของตัวเลขที่ป้อนเข้ามา:


            int sum = 0, num;
            printf("ป้อนตัวเลข (0 เพื่อออก): ");
            scanf("%d", &num);
            
            while (num != 0) {
                sum += num;
                printf("ป้อนตัวเลขอีกครั้ง: ");
                scanf("%d", &num);
            }
            
            printf("ผลรวมของตัวเลขที่ป้อน: %d\n", sum);

4. ข้อผิดพลาดที่พบบ่อย

เช่น การลืมเพิ่มค่าตัวแปรทำให้เกิด infinite loop

5. แบบทดสอบ

1.เขียนโปรแกรมคำนวณเลขคู่-เลขคี่
2.เขรยนโปรแกรมนับจำนวนตัวอักษร
3.เขียนโปรแกรมสร้างเมนูเลือกคำสั่ง
