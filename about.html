เกี่ยวกับ - คำสั่ง while loop ในภาษา C
หน้าแรก เกี่ยวกับ หลักสูตร ตัวอย่างโปรเจค ติดต่อ

คำสั่ง while loop คืออะไร?

คำสั่ง while loop ในภาษา C เป็นโครงสร้างควบคุมที่ช่วยให้โปรแกรมสามารถทำงานซ้ำ ๆ ได้จนกว่าจะเป็นไปตามเงื่อนไขที่กำหนด โดยจะตรวจสอบเงื่อนไขก่อนเสมอ หากเป็นจริงจะทำคำสั่งที่อยู่ภายในลูป

โครงสร้างของ while loop

while (เงื่อนไข) {
    คำสั่งที่ต้องการให้ทำซ้ำ;
}

ตัวอย่างการใช้ while loop

#include 

int main() {
    int i = 0;
    while (i < 5) {
        printf("%d\n", i);
        i++;
    }
    return 0;
}

ข้อดีของ while loop

  • เหมาะสำหรับกรณีที่จำนวนรอบของการทำซ้ำไม่แน่นอน
  • สามารถใช้ควบคุมเงื่อนไขได้อย่างยืดหยุ่น
  • ช่วยให้โค้ดกระชับและอ่านง่าย

ข้อควรระวัง

  • หากไม่มีการเปลี่ยนแปลงค่าของตัวแปรที่ใช้เป็นเงื่อนไข อาจเกิด Infinite Loop
  • ต้องแน่ใจว่าเงื่อนไขจะเป็น False ที่จุดหนึ่ง มิฉะนั้นโปรแกรมอาจค้าง

เปรียบเทียบ while loop กับ for loop

while loop ใช้ในกรณีที่จำนวนรอบไม่แน่นอน ขึ้นอยู่กับเงื่อนไขที่กำหนด
for loop ใช้เมื่อทราบจำนวนรอบที่แน่นอนล่วงหน้า
